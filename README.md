# Inertia React Setup with SSR
A modern web application starter template with Inertia.js, React, Server-Side Rendering (SSR), and Tailwind CSS.

## Tech Stack

- **Frontend Framework:** React with TypeScript
- **Styling:** Tailwind CSS

## Getting Started

### Prerequisites

- Node.js (v21 or higher)
- npm or yarn

### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/rpsohag/inertia-react-ssr.git inertia-react-ssr
    ```
2. go to inertia-react-ssr directory:

    ```bash
    cd inertia-react-ssr
    ```
3. Install composer packages:

    ```bash
    composer install
    ```
4. Install npm packages:

    ```bash
    npm install
    ```
5. Run this application:

     ```bash
    php artisan serve
    ```

6. Run npm dev server:

     ```bash
    npm run dev
    ```
